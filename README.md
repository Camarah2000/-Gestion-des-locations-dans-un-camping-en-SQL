Gestion des locations dans un camping (SQL)
Description du projet

Ce projet a pour objectif de mettre en place un système de gestion des locations dans un camping. À partir d’un énoncé fourni, j’ai conçu une solution complète permettant d’organiser et d’exploiter les données liées aux réservations.

Étapes de réalisation
Modélisation UML
Analyse du besoin à partir du texte fourni
Conception du modèle UML à l’aide d’un outil de modélisation
Conception de la base de données
Élaboration du modèle relationnel
Création des scripts SQL pour la base de données
Insertion des données dans les différentes tables
Exploitation des données
Rédaction de requêtes SQL pour extraire des informations pertinentes
